# Diwali Sales Analysis

This project performs Exploratory Data Analysis (EDA) on Diwali sales data to understand customer purchasing behavior and identify key revenue-driving segments.

## Dataset
The dataset contains over **11,000 retail transactions** including customer demographics, product categories, and purchase amounts.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow
1. Data cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. Visualization of customer behavior
4. Business insights generation

## Key Insights
- Female customers generated higher total sales than male customers.
- Customers aged **26–35 years** contributed the highest purchases.
- Top-performing states: **Uttar Pradesh, Maharashtra, Karnataka**.
- Most popular product categories: **Food, Clothing, Electronics**.
